# 📖 cash-tracker <a name="about-project"></a>

**drawing-app** is a simple React drawing application built using the HTML canvas component. Created to explore this component's functionality while practicing coding skills.

### Key Features <a name="key-features"></a>

- **Line drawing:** allows user to sketch simple things using lines of varying width and color.
- **Shadows:** allows user to draw with shadows.
- **Background color:** user may quickly change background color
